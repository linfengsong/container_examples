============
Installation
============


Stable release
--------------

To install Gitlab CI/CD Utils Project, run this command in your terminal:

.. code-block:: console

    $ pip install gitlab_cicd_utils

This is the preferred method to install Gitlab CI/CD Utils Project, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Gitlab CI/CD Utils Project can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://gitlab.com/linfengsong/gitlab_cicd_utils

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://gitlab.com/linfengsong/gitlab_cicd_utils/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Gitlab repo: https://gitlab.com/linfengsong/gitlab_cicd_utils
.. _tarball: https://gitlab.com/linfengsong/gitlab_cicd_utils/tarball/master
